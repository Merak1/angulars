ANGULAR

https://angularjs.org/

Es un framework de Javascript que se agrega via un tag "script" a una página HTML

AngularJS extiende los atributos HTML con Directivas (ng-directives), y vincula datos en el HTML con Expresiones

DIRECTIVAS

ng-app

Es la directiva que define una aplicación de Angular

ng-model

Es la directiva que vincula el valor de un tag de control de HTML (input, select, textarea) a los datos de la aplicación

Puede ser usada para validación de datos

o para darle clases de CSS a elementos HTML

ng-bind

Es la directiva que vincula un valor de los datos de la aplicación a la vista del HTML 


<script src="http://ajax.googleapis.com/ajax/libs/angularjs/1.3.14/angular.min.js"></script>
<body>
<div ng-app="">
  <p>Name: <input type="text" ng-model="name"></p>
  <p ng-bind="name"></p>
</div>
</body>

AngularJS empieza automáticamente cuando la página termina de cargar

La directiva ng-app le dice a AngularJS que el elemento <div> es el dueño de la aplicación.

La directiva ng-model vincula el valor del <input> a la variable "name" de la aplicación

La directiva ng-bind bincula el inner HTML del elemento <p> a la variable "name" de la aplicación

ng-init

Dentro del elemento que contenga ng-app, puedes inicializar variables utilizando la directiva ng-init

Expresiones

Las expresiones de Angular se escriben dentro de llaves dobles `{{ expresion }}` (igual que en Handlebarss) Las expresiones vinculos los datos al html de la misma manera que la directiva `ng-bind` El resultado estará exactamento donde la expresión esté escrita

Las expresiones funcionan parecido a Javascript. Pueden contener:

  • literales
  • operadores
  • variables

Salvo que no pueden contener

  • condicionales
  • loops



  
My first expression: {{ 5 + 5 }}

Las expresiones vinculan a los datos al HTML de la misma manera que la directiva ng-bind
