Bootstrap

https://getbootstrap.com/

Es una librería WEB (HTML, CSS, JS) que sirve para diseñar y prototipar aplicaciones responsivas

Utiliza componentes precontruidos y pluggins basados en JQuery

Puede instalarse via npm o accesando al cdn

Gris system

Bootstrap ocupa un sistema de layout para distintos tipos de dispositivos: celulares, tablets, laptops, pantallas

Utiliza un sistema de 12 columnas para definir cuánto ocupa cada elemento

Se utilizan prefijos para determinar para cada tamaño cuántas columnas ocupa: .col-xs-* .col-sm-* -.col-md-* .col-lg-*
